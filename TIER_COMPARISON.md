| **I6** | Threshold Bounds | Detection params stay in safe ranges |
| **I7** | No Self-Modification | RSI changes config, never code |
| **I8** | Rollback Guarantee | Every version preserved + restorable |

## Technical Implementation

Tier gating is a single environment variable:

```bash
# .env
AION_LICENSE_TIER=baseline     # or: intelligence, sentinel
```

The codebase is universal вҖ” all 67 core files deploy everywhere. The `TierConfig` module in `aionos/licensing/tier_config.py` gates features at the API layer. No code forks, no separate builds, no feature branches.

**Deployment cost is identical for all tiers.** Margin differences come purely from which features are unlocked.
